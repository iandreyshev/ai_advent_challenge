# Работа с задачами в TaskFlow

## Создание задач

### Как создать задачу
1. Откройте проект, в котором хотите создать задачу
2. Нажмите кнопку "+ Новая задача" в правом верхнем углу или используйте горячую клавишу 'N'
3. Заполните обязательные поля:
   - Название задачи
   - Описание (опционально, но рекомендуется)
4. Установите дополнительные параметры:
   - Приоритет (низкий, средний, высокий, критический)
   - Срок выполнения
   - Исполнитель
   - Метки (теги)
5. Нажмите "Создать"

### Проблемы при создании задач

#### Кнопка "Создать задачу" не работает
Возможные причины:
1. **Проблемы с JavaScript в браузере**
   - Отключены JavaScript
   - Конфликт с расширениями браузера (особенно блокировщики рекламы)
   - Устаревшая версия браузера

   **Решение:**
   - Проверьте, что JavaScript включен в настройках браузера
   - Попробуйте отключить расширения браузера
   - Обновите браузер до последней версии
   - Очистите кэш браузера (Ctrl+Shift+Del)

2. **Недостаточно прав доступа**
   - В бесплатном плане ограничение на количество задач (до 100)
   - Вы не добавлены в проект как участник с правами на создание задач

   **Решение:**
   - Проверьте лимиты вашего тарифного плана
   - Попросите администратора проекта предоставить вам необходимые права

3. **Технические проблемы**
   - Временные проблемы с сервером
   - Проблемы с сетевым подключением

   **Решение:**
   - Проверьте подключение к интернету
   - Обновите страницу (F5)
   - Попробуйте через несколько минут

#### Задача создается, но не появляется в списке
- Проверьте фильтры - возможно, задача скрыта из-за активных фильтров
- Обновите страницу
- Проверьте, что вы смотрите правильный проект
- Проблема может быть связана с синхронизацией (см. раздел "Синхронизация")

## Редактирование и удаление задач

### Как изменить задачу
- Кликните на название задачи, чтобы открыть детали
- Нажмите иконку редактирования (карандаш)
- Внесите изменения и сохраните

### Как удалить задачу
- Откройте задачу
- Нажмите иконку корзины в правом верхнем углу
- Подтвердите удаление

**Внимание:** Удаленные задачи перемещаются в корзину и могут быть восстановлены в течение 30 дней.

## Горячие клавиши
- `N` - создать новую задачу
- `E` - редактировать выбранную задачу
- `Del` - удалить выбранную задачу
- `/` - быстрый поиск
- `Ctrl+K` - командная палитра
