# Синхронизация в TaskFlow

## Как работает синхронизация

TaskFlow автоматически синхронизирует данные между всеми вашими устройствами:
- Веб-версия (браузер)
- Мобильное приложение (iOS/Android)
- Desktop приложение (Windows/Mac/Linux)

Синхронизация происходит в реальном времени при наличии подключения к интернету.

## Проблемы с синхронизацией

### Изменения не синхронизируются между устройствами

**Причина 1: Проблемы с интернет-подключением**
- Проверьте подключение к интернету на обоих устройствах
- Убедитесь, что нет ограничений в корпоративной сети (порты, firewall)
- Попробуйте переключиться между WiFi и мобильным интернетом

**Причина 2: Устаревшая версия приложения**
- Обновите мобильное/desktop приложение до последней версии
- Обновите страницу в браузере (Ctrl+F5 для полного обновления)
- Проверьте версию приложения в Настройки → О программе

**Причина 3: Конфликт синхронизации**
- Происходит, когда одна и та же задача редактируется одновременно на разных устройствах
- Система попытается объединить изменения автоматически
- В случае конфликта вы увидите уведомление с просьбой выбрать версию

**Причина 4: Кэш браузера**
- Очистите кэш браузера
- В Chrome: Ctrl+Shift+Del → Очистить данные за все время
- Перезайдите в аккаунт

**Причина 5: Офлайн-режим**
- Проверьте, не включен ли офлайн-режим
- В мобильном приложении: Настройки → Синхронизация → Автоматическая синхронизация (должна быть включена)

### Как принудительно синхронизировать данные

**В веб-версии:**
- Обновите страницу (F5)
- Используйте Ctrl+Shift+R для жесткой перезагрузки

**В мобильном приложении:**
- Потяните список задач вниз (pull-to-refresh)
- Или зайдите в Настройки → Синхронизация → Синхронизировать сейчас

**В desktop приложении:**
- Меню → Файл → Синхронизировать
- Или используйте горячую клавишу Ctrl+R

### Данные синхронизируются с задержкой

Нормальная задержка синхронизации составляет 1-2 секунды.

Если задержка больше:
1. Проверьте скорость интернета
2. Убедитесь, что сервера TaskFlow доступны (статус на status.taskflow.com)
3. Попробуйте выйти и снова войти в аккаунт
4. Проверьте, нет ли больших вложений в задачах (они синхронизируются дольше)

### Потеря данных при синхронизации

Это не должно происходить. Если вы заметили потерю данных:
1. Проверьте корзину - возможно, данные были случайно удалены
2. Проверьте историю изменений задачи
3. Немедленно свяжитесь с поддержкой support@taskflow.com
4. Не вносите новых изменений до решения проблемы

## Офлайн-режим

TaskFlow поддерживает работу без интернета:
- Все данные кэшируются локально
- Вы можете просматривать и редактировать задачи офлайн
- Изменения будут синхронизированы при восстановлении подключения

**Ограничения офлайн-режима:**
- Нельзя пригласить новых пользователей
- Нельзя загрузить новые вложения
- Поиск работает только по локальному кэшу
- Некоторые функции интеграций недоступны

## Настройки синхронизации

**Частота синхронизации:**
- Настройки → Синхронизация → Частота проверки обновлений
- Рекомендуется: Автоматически (по мере изменений)
- Для экономии трафика: Каждые 5/15/30 минут

**Что синхронизировать:**
- Все проекты (по умолчанию)
- Только избранные проекты (для экономии места)
- Настройки → Синхронизация → Выбрать проекты

**Синхронизация вложений:**
- По умолчанию вложения синхронизируются автоматически
- Можно отключить для экономии трафика
- Настройки → Синхронизация → Загружать вложения: Только по WiFi / Всегда / Вручную
