# День 1: iOS чат с ChatGPT

Простое iOS-приложение для общения с ChatGPT через OpenAI API.

## Возможности

- Чат-интерфейс с GPT-4o-mini
- Персонализированный системный промпт (друг-программист в стиле фильмов "Брат 2" и "Жмурки")
- Парсинг структурированных ответов (JSON + текст)
- История сообщений в сессии

## Технологии

- **SwiftUI** — пользовательский интерфейс
- **OpenAI Chat Completions API** — модель gpt-4o-mini
- **URLSession** — HTTP запросы
- **Combine** — реактивность

## Структура

```
AIAdventChallengeDay1/
├── ContentView.swift    # Основной UI и логика чата
├── Secrets.swift        # API ключи (gitignored)
└── Assets/
```

## Запуск

1. Создайте файл `Secrets.swift`:
```swift
enum Secrets {
    static let openAIKey = "sk-your-api-key"
}
```

2. Откройте проект в Xcode и запустите на симуляторе

## Особенности реализации

- Системный промпт задаёт характер собеседника
- Ответ модели парсится: JSON на первой строке, затем текстовое объяснение
- Поддержка истории диалога для контекста
