# День 8: Token Monitor

iOS-приложение для мониторинга использования токенов в запросах к LLM.

## Возможности

- Подсчёт токенов и слов в запросе/ответе
- Отслеживание времени ответа
- Обработка ошибок превышения лимита токенов
- Примеры запросов для тестирования

## Технологии

- **SwiftUI** — пользовательский интерфейс
- **OpenAI Responses API** — endpoint `/v1/responses`
- **URLSession** — HTTP запросы

## Отслеживаемые метрики

| Метрика | Описание |
|---------|----------|
| Токены запроса | Количество токенов в промпте |
| Токены ответа | Количество токенов в ответе |
| Слова | Количество слов (отдельный подсчёт) |
| Время | Секунды до получения ответа |

## Структура

```
AIAdventChallengeDay8/
├── ContentView.swift    # UI мониторинга
├── Secrets.swift        # API ключи (gitignored)
└── Assets/
```

## Запуск

1. Добавьте API ключ в `Secrets.swift`
2. Откройте в Xcode и запустите

## Особенности

- Кнопка "Вставить пример" для быстрого тестирования
- Обработка `tokenLimitError` и `badStatus`
- Функция `wordCount()` для подсчёта слов
