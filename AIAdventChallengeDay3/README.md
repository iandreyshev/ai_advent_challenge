# День 3: Карьерный опрос по IT

iOS-приложение с многошаговым опросом для определения подходящей IT-профессии.

## Возможности

- Динамическая генерация вопросов на основе предыдущих ответов
- Максимум 5 вопросов для определения профиля
- LLM выступает как карьерный консультант
- Финальное резюме с рекомендациями и шагами

## Технологии

- **SwiftUI** — пользовательский интерфейс
- **OpenAI Chat Completions API** — gpt-4o-mini
- **URLSession** — HTTP запросы

## Как работает

1. Пользователь отвечает на вопросы о своих интересах и навыках
2. LLM генерирует следующий вопрос на основе истории ответов
3. После 5 вопросов формируется итоговое резюме
4. Резюме содержит рекомендуемые направления и 2-4 конкретных шага

## Запуск

1. Добавьте API ключ в `Secrets.swift`
2. Откройте в Xcode и запустите

## Особенности

- Прогресс-индикатор "Вопрос X из 5"
- Две API-инструкции: для генерации вопросов и подведения итогов
- Многострочное поле ввода для развёрнутых ответов
