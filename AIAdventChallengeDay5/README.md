# День 5: Чат с выбором персоны

iOS-приложение с возможностью выбора характера собеседника.

## Возможности

- Три персоны на выбор: "Программист", "Весёлый водитель", "Моя девушка"
- Сохранение контекста диалога через OpenAI Responses API
- Копирование сообщений в буфер обмена
- Смена персоны без сброса истории

## Технологии

- **SwiftUI** — пользовательский интерфейс
- **OpenAI Responses API** — сохраняет контекст через `previous_response_id`
- **UIKit** — интеграция с UIPasteboard

## Персоны

| Персона | Описание |
|---------|----------|
| Программист | Технический, помогает с кодом |
| Весёлый водитель | Разговорчивый, с юмором |
| Моя девушка | Заботливая, эмоциональная |

## Структура

```
AIAdventChallengeDay5/
├── ContentView.swift    # UI чата и выбор персоны
├── Secrets.swift        # API ключи (gitignored)
└── Assets/
```

## Запуск

1. Добавьте API ключ в `Secrets.swift`
2. Откройте в Xcode и запустите

## Особенности реализации

- OpenAI Responses API использует `response_id` для связи сообщений
- Персона влияет на системный промпт
- Enum для управления типами персон
