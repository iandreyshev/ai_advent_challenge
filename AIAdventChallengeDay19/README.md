# День 19: Чат с RAG и Yandex AI

iOS-приложение для общения с LLM с использованием сжатия контекста.

## Возможности

- Чат с Yandex AI Studio
- Автоматическое сжатие истории диалога
- Подсчёт токенов (input/output/total)
- Сравнение режимов с/без компрессии
- Демо-режим для тестирования

## Технологии

- **SwiftUI** — пользовательский интерфейс
- **Yandex AI Studio API** — языковая модель
- **URLSession** — HTTP запросы
- **FileManager** — персистентность

## Как работает сжатие

```
[4 сообщения] → LLM Summary → [Summary + последние 3]
```

- Каждые 4 сообщения создаётся краткое summary
- Локальный контекст: summary + 3 последних сообщения
- Экономия токенов до 70%

## Запуск

1. Добавьте Yandex API ключ в `AppSecrets.swift`
2. Откройте в Xcode и запустите

## Метрики

- Время ответа (секунды)
- Токены запроса/ответа
- Количество слов
- Экономия при сжатии
